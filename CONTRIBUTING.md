# Contributing Guidelines

Thank you for your interest in contributing to **CBA for Platform Workers GPT**.

This project welcomes contributions from:
- Labor-law researchers
- Union representatives
- Platform companies
- Policy analysts
- Academic experts
- Open-source collaborators

### 1. Reporting Issues
Open an Issue for:
- Errors in CBA summaries  
- Missing CBAs (Europe, last 5 years)  
- Incorrect legal interpretations  
- Typos or formatting issues  
- Feature requests  

### 2. Submitting Improvements
You may contribute:
- Additional CBA analyses  
- Data structure improvements  
- New clause templates  
- Enhancements to prompt instructions  
- Comparative tables  
- Updated jurisprudence summaries  

### 3. Pull Request Process
1. Fork the repository  
2. Create a feature branch  
3. Commit your updates  
4. Submit a Pull Request  

Ensure that:
- Citations are included  
- Tone remains academic-technical  
- European labor-law terminology is correct  
- No personal or confidential information is included  

### 4. Citation Requirements
All contributions **must cite sources**, preferably:
- ILO Digital Labour Platform  
- GDPoweR Project  
- Official CBAs (public versions)  
- Academic publications  
- EU and national legislation
